Project Title:
Netflix Homepage Clone

Description:
This project is a frontend clone of the Netflix homepage developed using HTML and CSS. The main objective of this project is to replicate the user interface and design of the Netflix landing page, focusing on layout, styling, and visual presentation. It demonstrates the ability to build responsive and visually appealing web pages.

Features:
Netflix-style homepage design
Header with logo and navigation
Hero section with background image
Content sections similar to Netflix layout
Footer with links and information

Technologies Used:
HTML
CSS


Project Workflow:
The project focuses on designing the homepage structure using HTML and styling it using CSS to closely match the original Netflix interface. Different sections such as header, banner, and content rows are created to simulate a real streaming platform UI.

My Role:
This project was developed independently by me. I handled the complete design and implementation of the frontend, including layout structuring and styling.
